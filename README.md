# 收支小本法律文档与用户支持

这里是收支小本公开使用的文档中心，包含隐私政策、用户协议和用户支持页面。

如果你是从 App 内网页或 GitHub Pages 进入这里，可以直接查看下面三个页面：

## 页面入口

- [隐私政策](privacy-policy.html)
- [用户协议](user-agreement.html)
- [用户支持](support.html)

## 关于收支小本

收支小本是一款面向个人与家庭场景的记账应用，支持交易记录、分类管理、账户管理、预算查看，以及登录、同步、备份恢复和协作等相关能力。

这个目录主要用于公开展示以下内容：

- 收支小本隐私政策
- 收支小本用户协议
- 收支小本用户支持信息

## 使用说明

- 如果你只是想尽快解决问题，建议优先查看“用户支持”页面。
- 如果你想了解数据处理方式、账号处理和删除边界，建议查看“隐私政策”页面。
- 如果你想确认登录、同步、购买恢复和协作使用规则，建议查看“用户协议”页面。
- 如果你是从 App 内打开网页，这些页面会以手机阅读体验为主进行展示。
